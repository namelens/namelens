# NameLens v0.2.2 Release Notes

Release date: 2026-02-20

## Highlights

v0.2.2 is a model refresh: Anthropic and OpenAI model tiers updated to current
releases, and OpenAI's reasoning tier is now enabled for deep analysis
workloads.

## Model Updates

| Provider  | Tier        | Before                     | After                     |
| --------- | ----------- | -------------------------- | ------------------------- |
| Anthropic | `default`   | claude-sonnet-4-5-20250929 | claude-sonnet-4-6         |
| Anthropic | `reasoning` | claude-sonnet-4-5-20250929 | claude-sonnet-4-6         |
| Anthropic | `fast`      | claude-3-5-haiku-20241022  | claude-haiku-4-5-20251001 |
| OpenAI    | `reasoning` | (not set)                  | o3                        |

### Why These Changes

**Anthropic:** `claude-sonnet-4-6` is the current Sonnet release, offering
improved instruction following and structured output reliability. The fast tier
moves to `claude-haiku-4-5-20251001`, the latest Haiku release.

**OpenAI `o3` reasoning tier:** OpenAI's o-series models are purpose-built for
multi-step reasoning tasks. For `--depth=deep` workflows (brand review, deep
analysis), `o3` delivers significantly higher quality than standard chat models.
The model is only invoked when `--depth=deep` is explicitly requested, so
default usage and cost are unaffected.

## Configuration

To apply these model updates to an existing install:

```bash
# Re-run setup wizard (preserves all other config)
namelens setup --provider anthropic --api-key $ANTHROPIC_KEY
namelens setup --provider openai --api-key $OPENAI_KEY

# Or edit directly
$EDITOR ~/.config/namelens/config.yaml
```

Minimal diff for manual update:

```yaml
namelens-anthropic:
  models:
    default: claude-sonnet-4-6        # was: claude-sonnet-4-5-20250929
    reasoning: claude-sonnet-4-6      # was: claude-sonnet-4-5-20250929
    fast: claude-haiku-4-5-20251001   # was: claude-3-5-haiku-20241022

namelens-openai:
  models:
    reasoning: o3                     # was: (not set)
```

## Upgrade Notes

No breaking changes. Existing configs with the old Anthropic model names remain
valid — those models are still accessible. The OpenAI reasoning tier is purely
additive; workloads that don't use `--depth=deep` are unaffected.
